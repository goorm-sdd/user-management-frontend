# 회원관리 서비스

---

### 📌 프로젝트 개요

- **프로젝트 명** : 회원관리 서비스 
- **진행 기간** : 2025년 5월 27일 ~ 2025년 6월 16일
- **기획 배경 및 목적** : 회원관리 기능 중심의 웹 서비스 개발
- **기획 기간** : 2025 5월 27일 ~ 2025년 6월 2일

#### 주요 기능 요약

| 기능           | 설명                              |
| -------------- | -------------------------------- |
| 회원가입       | 사용자 정보 입력, 유효성 검사 포함 |
| 로그인/로그아웃 | 세션 또는 JWT 기반 로그인 상태 유지 |
| 회원정보 조회/수정 | 프로필 페이지, 정보 수정 폼 제공    |
| 회원 삭제      | 본인 계정 삭제 기능, 삭제 확인 절차 포함 |
| 관리자 기능    | 전체 회원 목록 관리, 검색/정렬/필터 제공 |

---

### 🎯 프로젝트 목표 및 산출물

- **최종 목표** : 사용자 및 관리자 기능이 포함된 완성도 높은 회원관리 웹 서비스 구현
